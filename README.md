🧁 Bakery Planner — Premium Management Suite
Bakery Planner is a high-performance, standalone desktop application designed for professional bakery owners and managers. Built with a focus on data integrity, visual analytics, and a premium user experience, it serves as the "brain" of your bakery operations.

🚀 Key Features
📊 Smart Dashboard: Real-time visual analytics with revenue bar charts, best-seller rankings, and automated business alerts.
🍞 Production Optimizer: A smart engine that calculates how many batches of a product you can bake based on your current ingredient levels.
📦 Inventory & Waste Tracking: Detailed ingredient management with "Low Stock" alerts and a disposal system for expiring waste.
🧑‍🍳 Interactive Recipe Builder: Create and manage complex recipes with real-time ingredient requirement calculation.
📋 Order & Sales Management: Track the full lifecycle of customer orders from "Pending" to "Completed" and maintain detailed financial sales reports.
👥 Customer CRM: Manage your customer base and view their lifetime order history and spending habits.
🔔 Smart Notifications: A proactive alert system that identifies expired products and inventory shortages the moment you start the app.
🛠️ Tech Stack
Core: Java 21 (Object-Oriented Architecture)
GUI: Java Swing (Fully custom-styled dark mode with smooth transitions)
Persistence: SQLite via JDBC (Reliable local-first data storage)
Design Pattern: Manager-Facade Pattern for clean, maintainable logic.
🎨 Design Aesthetic
The application features a modern, premium dark-mode aesthetic with:

Gold & Slate Palette: Curated colors for a high-end professional feel.
Dynamic Visuals: Custom-painted bar charts and status indicators.
Intuitive Navigation: An animated sidebar and streamlined workflows to minimize clicks.
📦 Installation & Setup
Ensure you have JDK 21 installed.
Include the sqlite-jdbc.jar in your classpath.
Run Main.java to initialize the database and start the dashboard.
